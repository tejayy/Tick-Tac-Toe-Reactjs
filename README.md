## Overview

This project is a web-based Tic Tac Toe game created using ReactJS. It allows two players to take turns, marking spaces on a 3x3 grid. The game declares a winner when a player successfully aligns three marks horizontally, vertically, or diagonally.

## Features

- Two-player turn-based gameplay
- Interactive UI with React components
- Simple and clean design

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/tejayy/Tick-Tac-Toe-Reactjs.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Tick-Tac-Toe-Reactjs
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the development server:

   ```bash
   npm start
   ```

2. Open your browser and go to [http://localhost:3000](http://localhost:3000) to play the Tic Tac Toe game.

3. Follow the on-screen instructions to take turns and enjoy the game.
